Speech Emotion Recognition (Beyond_Words 3.0)

Description:

Speech Emotion Recognition (Beyond_Words 3.0) is a system designed to analyze human emotions from speech. Using the SUBESCO dataset, the model employs Convolutional Neural Networks (CNNs) to achieve an accuracy of 99.38%. This project focuses on understanding human emotions through speech analysis and can be applied in areas such as mental health, customer service, and entertainment. The system efficiently handles large datasets and optimizes model performance for accurate emotion classification.
This research was presented at the INDICON (IEEE India Council International Conference) 2024 at IIT Kharagpur, where it received valuable feedback and recognition from experts in the field.

Features:

  • High Accuracy(99.38%)
  • Speech Emotion Detection
  • Efficient Dataset Handling
  • Optimized Model Performance
  • Real-World Applications

  • TensorFlow
  • Keras
  • Convolutional Neural Networks (CNN)
  • SUBESCO Dataset

Beyond Words(2.0)

Description:

Beyond Words is an emotion detection application powered by machine learning. It analyzes facial expressions from pre-recorded video files and emotion from voice in audio files to classify emotions into seven categories: angry, disgust, fear, happy, neutral, sad, and surprise. The app offers a user-friendly interface built with Streamlit, providing a seamless experience for users to upload and process video or audio files for emotion detection. The models are trained on FER2013 for facial emotion detection and RAVDESS for voice emotion recognition.

Features:

  • Emotion Detection from Video
  • Emotion Detection from Audio
  • User-Friendly Interface
  • Face Detection
  • Pre-trained CNN Model for Video
  • Pre-trained RNN Model for Audio
  • Separate Audio and Video Sections

Technologies:

  • Python
  • Keras
  • TensorFlow
  • OpenCV
  • Streamlit
  • Librosa
  • NumPy
  • Matplotlib

Jarvis Assistant

Description:

Jarvis Assistant is a Python-based voice assistant that utilizes speech recognition, pyttsx3, and the Cohere API to perform a variety of tasks and provide information to users through voice commands. The assistant is designed to handle task automation, provide weather updates, manage emails, and solve complex coding issues through voice processing. Jarvis can understand and respond to user queries effectively. The project was also awarded third prize in a project competition.
